# CSS Advanced Project

This project is part of the HTML & CSS curriculum at ALX Software Engineering.

## Objectives

- Practice advanced CSS styling techniques
- Implement a web page design using a Figma prototype
- Ensure HTML and CSS code are valid using W3C validators
